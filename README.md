# OrangeHRM Login Module - Manual Testing

## Project Overview

This project demonstrates manual testing of the OrangeHRM Demo application by designing comprehensive test cases for the Login module. The objective is to verify the functionality, usability, validation, and security of the login feature using industry-standard QA practices.

- Application: OrangeHRM Demo
